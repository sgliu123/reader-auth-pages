🚀 快速开始（5分钟上手）

  • 第一步：导入书籍：点击顶部"📚 书籍"按钮 → 选择"📄 导入书籍" → 从电脑选择TXT、PDF、Word(.docx)或EPUB格式的电子书
  • 第二步：开始阅读：书籍导入后自动显示内容，点击"▶️ 播放"按钮即可语音朗读
  • 第三步：翻页浏览：点击"◀ 上一页"/"▶ 下一页"或使用底部进度条跳转
  • 第四步：添加笔记：用鼠标选中任意文字 → 在弹出窗口中输入笔记 → 点击"保存笔记"
  • 第五步：查看目录：点击"📑 目录"按钮快速跳转到任意章节

📖 导入书籍详解

  • TXT格式：纯文本文件，导入速度最快，支持自动分句朗读
  • PDF格式：保留原始排版和图片，支持逐页浏览和句子级定位
  • Word(.docx)格式：保留文档结构和图片，自动提取文本内容
  • EPUB格式：电子书标准格式，支持章节导航和翻页动画
  • 导入提示：大文件导入可能需要几秒钟，请耐心等待加载提示消失

🎙️ 语音朗读功能

  • 开始朗读：点击顶部或底部的"▶️ 播放"按钮，从当前位置开始朗读
  • 暂停/继续：朗读中再次点击"⏸ 暂停"按钮可暂停，再次点击继续
  • 停止朗读：点击"⏹ 停止"按钮完全停止朗读
  • 从选中处播放：选中文字后点击"从选中处播放"，从该位置开始朗读
  • 循环播放：选中文字后点击"循环播放选中"，重复朗读该段内容
  • 音色选择：在"⚙ 设置"面板中，滚动选择喜欢的语音音色，点击即可应用
  • 语速调节：在设置面板拖动"语速"滑块（0.5x-2.0x），实时生效
  • 音量调节：在设置面板拖动"音量"滑块（0-100%）

⚙️ 阅读设置详解

  • 夜间模式：点击"🌙 切换夜间模式"，深色背景保护眼睛，适合夜间阅读
  • 字体大小：13档可选（12px-36px），点击数字按钮即时生效
  • 字间距：拖动滑块调节字符间距（0-5px），让文字更疏朗或紧凑
  • 行间距：拖动滑块调节行高（1.0-3.0），改善阅读舒适度
  • 背景主题：10种配色可选，包括纸张白、护眼绿、夜间蓝、复古 sepia 等
  • 字体样式：思源黑体（默认推荐）、宋体、楷体、Arial 四种可选
  • 设置自动保存：所有设置更改后自动保存，下次打开自动恢复

🤖 AI 总结功能（API 模式 - 推荐）

💡 为什么要用 API 模式？ API 模式调用云端大模型（如 DeepSeek、Qwen 等），总结质量远超本地算法，能深度理解文本语义、逻辑关系和隐含观点。

第一步：注册 SiliconFlow（硅基流动）账号

  1. 打开浏览器，访问官网：https://cloud.siliconflow.cn
  2. 点击右上角"登录"按钮，选择"短信登录"或"邮箱登录"
  3. 按提示填写手机号/邮箱，获取验证码，完成注册
  4. 新用户注册即送免费额度，部分模型永久免费

第二步：创建 API Key

  1. 登录后，点击左侧菜单"API密钥"（或在首页找到"API密钥"入口）
  2. 点击"新建API密钥"按钮
  3. 输入密钥名称（如"阅读器"），点击确认创建
  4. 立即复制生成的 API Key（格式如：sk-xxxxxxxxxx），此密钥只显示一次，务必保存好！
  5. 如果忘记保存，可以删除旧密钥重新创建一个新的

第三步：在阅读器中配置 API

  1. 在阅读器中点击"🛠️ 工具" → "🤖 AI总结"，打开总结弹窗
  2. 勾选"使用 API 调用 AI 总结"复选框
  3. 在"API Key"输入框中，粘贴刚才复制的密钥（sk-开头的那串字符）
  4. "API 地址"保持默认：https://api.siliconflow.cn/v1
  5. "模型 ID"保持默认：Qwen/Qwen2.5-7B-Instruct（免费模型，速度快）
  6. 如需更强效果，可更换为：deepseek-ai/DeepSeek-V2.5（需有额度）
  7. 配置完成后，点击"🚀 生成总结"即可调用云端 AI

第四步：验证配置是否成功

  1. 确保已导入书籍且内容不为空
  2. 选择"本章总结"或"全文总结"
  3. 点击"🚀 生成总结"
  4. 如果看到"🤖 正在生成总结，请稍候..."，说明配置正确，正在调用 API
  5. 如果报错，请检查：API Key 是否复制完整、网络是否正常、模型 ID 是否正确

❓ 常见问题解答（FAQ）

  • Q: 导入 PDF 后文字显示乱码？
    A: 请确保 PDF 是文字版而非扫描版（图片版）。扫描版 PDF 无法提取文字，建议先用 OCR 软件转换。
  • Q: 语音朗读不流畅或跳过内容？
    A: 尝试调慢语速（0.8x-1.0x），或检查文本中是否有过多的特殊符号、英文混排。
  • Q: 笔记突然消失了？
    A: 笔记保存在浏览器本地存储中，清除浏览器数据会导致笔记丢失。建议定期导出备份。
  • Q: 换电脑后书籍和笔记还在吗？
    A: 不在。所有数据保存在当前设备的浏览器中，换设备需重新导入书籍。笔记可通过导出功能备份转移。
  • Q: API 总结报错"网络请求失败"？
    A: 检查网络连接；确认 API Key 正确且未过期；确认 API 地址和模型 ID 填写无误；尝试更换模型。
  • Q: 可以离线使用吗？
    A: 基础功能（阅读、朗读、笔记）完全离线可用。翻译和 API 总结需要网络连接。
  • Q: 支持手机/平板使用吗？
